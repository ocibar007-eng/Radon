# REFERÊNCIAS RÁPIDAS — USG ABDOME PEDIÁTRICO (NORMAS LOCAIS)

Este arquivo existe para **evitar pesquisa na internet** durante a geração do laudo e reduzir tempo de processamento.

Regra: usar estes valores apenas como **apoio** e sempre correlacionar com o contexto clínico, técnica e biotipo.

## Rim — comprimento esperado (apoio; Rosenbaum 1984)

Para crianças:

- > 1 ano:
  `Comprimento renal (cm) = 6,79 + 0,22 × idade (anos)`
- < 1 ano:
  `Comprimento renal (cm) = 4,98 + 0,155 × idade (meses)`

Observação: usar na **auditoria interna** se precisar justificar “compatível com faixa etária”, evitando inserir idade numérica no laudo quando o template proibir.
